# 余興クイズシステム

JC大懇親会・余興向けのリアルタイムクイズシステム。

## 画面構成

| URL | 役割 | 使用者 |
|---|---|---|
| `/` | 参加者ページ（QRから開く） | 参加者（スマホ） |
| `/screen.html` | スクリーン表示（プロジェクター投影） | 司会者PC |
| `/admin.html` | 管理者操作パネル | 司会者PC（別タブ） |

## セットアップ

```bash
cd D:\jc\yoko_system
npm install
```

## 起動（ローカルテスト用）

```bash
npm start
# → http://localhost:3000 で起動
```

## インターネット公開（ngrok：当日の簡単な方法）

```bash
# Node.js起動後、別ターミナルで:
npx ngrok http 3000
# → https://xxxx.ngrok.io のようなURLが発行される
# スクリーンにそのURLのQRが自動表示される
```

## インターネット公開（Railway：永続URLが欲しい場合）

1. GitHubにこのフォルダをpush
2. https://railway.app でNew Project → Deploy from GitHub
3. 自動デプロイ完了 → URLが発行される
4. 以降はそのURLを使い回せる

## 当日の操作手順

1. `npm start` でサーバー起動
2. `screen.html` をプロジェクターに全画面表示（F11）
3. `admin.html` を手元のタブで開く
4. QRコードが表示されるので参加者にスキャンさせる
5. 問題ごとに:
   - 管理画面で「問題N・制限時間X秒・▶開始」ボタン
   - スライドで問題を見せる（このシステムは問題文を管理しない）
   - 参加者がABCDを選択 → リアルタイム集計がスクリーンに表示
   - 正解ボタン（A/B/C/D）を押す → スクリーンに正解＋ランキング表示
6. 全問終了後「🏆最終結果を表示」ボタン

## 採点方式

- 正解: **500〜1000pt**（回答速度に応じて変動）
- 最速回答: 1000pt / 最後の1秒: 約500pt
- 不正解・未回答: 0pt
- 10問合計: 最大10,000pt
