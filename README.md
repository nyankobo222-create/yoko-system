# JC余興クイズシステム

JC大懇親会・余興向けのリアルタイムクイズシステム。  
参加者は **代表者** と **会場参加者** の2役に分かれ、それぞれ別のルールで得点を競います。

---

## 画面構成

| URL | 役割 | 使用者 |
|---|---|---|
| `/` | 参加者ページ（QRから開く） | 代表者・会場参加者（スマホ） |
| `/screen.html` | スクリーン表示（プロジェクター投影） | 司会者PC |
| `/admin.html` | 管理者操作パネル（パスワード必要） | 司会者スマホ or 別タブ |

---

## 2つのロール

### 🏆 代表者
- QRをスキャンして「代表者として参加」を選択・名前を入力
- 問題が表示されたら A〜D の選択肢を1つタップ
- **回答速度が得点に直結**（速いほど高得点）

### 👥 会場参加者
- QRをスキャンして「会場として参加」を選択・名前を入力
- 問題表示中に「正解しそうな代表者」を予測してタップ（**複数選択可**）
- カウントダウン後は予測変更不可
- 的中人数に応じてポイント加算・外れ人数に応じて減算

---

## ゲームの流れ

```
ロビー → プレビュー → 3・2・1カウント → 出題中 → 正解発表 → 次の問題へ
```

| フェーズ | スクリーン表示 | 代表者スマホ | 会場参加者スマホ |
|---|---|---|---|
| **ロビー** | QRコード・参加者数 | 名前入力→待機 | 名前入力→待機 |
| **プレビュー** | 「予測中...」アニメ | 待機（問題は見えない） | 問題文表示→代表者を選択 |
| **カウントダウン** | 3・2・1 | カウントダウン | 選択ロック |
| **出題中** | 回答グラフ・予測状況 | 問題文＋ABCD表示→回答 | 選択済み表示（変更不可） |
| **正解発表** | 正解・スピードランキング | 正解/不正解＋得点 | 的中結果＋得点 |

## 採点方式

### 代表者
| 状況 | 得点 |
|---|---|
| 正解（最速） | 1,000 pt |
| 正解（残り半分の時間） | 750 pt |
| 正解（ギリギリ） | 500 pt |
| 不正解・未回答 | 0 pt |

### 会場参加者（デフォルト値、管理画面で変更可）
| 状況 | 得点 |
|---|---|
| 的中1人につき | +300 pt |
| 外れ1人につき | −100 pt |
| 合計がマイナスになる場合 | 0 pt（フロア制限） |

## 当日の操作手順

1. RailwayのURLで `/admin.html` を開く → **パスワード入力**
2. `/screen.html` をプロジェクターで**全画面表示**（F11）
3. 参加者にQRコードをスキャンさせる（名前入力→ロール選択）
4. 「ルール説明画面」を確認・編集して「スクリーンに表示」
5. **テスト問題**で動作確認（得点に入らない）
6. 問題ごとに以下を繰り返す:
   - 問題バンクから問題をクリックして選択（または手入力）
   - `📺 表示` → 会場参加者が予測中（代表者は待機）
   - `⏱ スタート` → 3・2・1カウントダウン → 全員同時に問題表示・回答開始
   - 回答状況を確認してから、正解ボタン（A / B / C / D）を押す
7. 全問終了後「🏆 最終結果を表示」ボタン
8. **ランキング演出パネル**で順位を1人ずつ発表

## 管理画面の主な機能

| 機能 | 説明 |
|---|---|
| パスワード認証 | ページを開いた瞬間に認証が必要（URLが漏れても安全） |
| 問題バンク | 問題を事前登録・JSON一括インポート・エクスポート対応 |
| 会場採点設定 | 的中/外れポイントをリアルタイムで変更可 |
| 参加者キック | 誤参加・重複をその場で退場させる（「✕退」ボタン） |
| スコア詳細 | 各参加者の問題ごとの回答・得点内訳を展開表示 |
| テスト問題 | 得点に入らない動作確認モード |
| ランキング演出 | 中間・最終を1人ずつスクリーンに表示（MC演出向け） |
| スマホ対応 | 管理画面もスマホ表示に最適化済み |

## 注意事項

- 全参加者データは**メモリ上**に保持されます。サーバー再起動でリセットされます
- 切断後 **2分以内** の再接続ならスコアは自動復元されます
- 接続状態は参加者スマホの右上インジケーター（●）で確認できます
- `sample_questions.json` を管理画面の「JSON読込」でインポートするとデモ用8問が使えます
